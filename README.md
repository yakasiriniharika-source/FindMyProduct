# 🛍️ Find My Product

## 📌 Description
Find My Product is a web application that helps users find electronic products available in nearby stores with real-time stock updates.

## 🚀 Features
- 🔍 Search products easily
- 📍 Find nearby stores
- 🗺️ Map integration (Leaflet)
- 📦 Real-time stock availability
- 🔔 Notify when product is out of stock

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
